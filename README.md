# 授業課題管理アプリ

## 概要

授業と課題を管理するWebアプリです。  
月曜〜土曜、1限〜6限の時間割表に授業を登録し、各授業に課題を追加できます。

GitHub Pagesで公開できるように、HTML・CSS・JavaScriptのみで作成しています。

## 主な機能

- 授業登録
- 課題登録
- 授業削除
- 時間割表の表示
- 授業ごとの課題一覧表示
- localStorageによるデータ保存

## 授業登録機能

曜日、時限、授業名を入力して授業を登録できます。

同じ曜日・時限にすでに授業がある場合は、重複登録を防ぐためエラーを表示します。

## 課題登録機能

登録済みの授業を選択し、その授業に課題を追加できます。

授業が登録されていない場合や、課題名が空欄の場合はエラーを表示します。

## 授業削除機能

登録済みの授業を選択して削除できます。

授業を削除すると、その授業に登録されている課題も同時に削除されます。

## 時間割表示機能

月曜〜土曜、1限〜6限の時間割表を表示します。

登録した授業は、対応する曜日・時限のマスに表示されます。  
また、授業ごとの課題数も表示されます。

## データ保存

データはブラウザの `localStorage` に保存されます。  
そのため、ページを更新しても同じブラウザであれば登録内容が残ります。

ただし、別の端末や別のブラウザとはデータを共有できません。


## データ構造

```javascript
[
  {
    "day": "月曜",
    "period": "1限",
    "name": "情報数学",
    "assignments": ["レポート1", "レポート2"]
  }
]
```

## ファイル構成

```text
homework/
├─ index.html
├─ README.md
└─ docs/
   └─ test-report.md
```

## 使い方

1. 曜日・時限・授業名を入力して授業を登録する
2. 登録済みの授業を選択して課題を登録する
3. 時間割表で授業と課題数を確認する
4. 不要な授業は削除する

## 注意点

- データはブラウザ内に保存されます
- ブラウザのデータを削除すると登録内容も消えます
- ログイン機能やクラウド保存機能はありません
