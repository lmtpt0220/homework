# 授業課題管理アプリ

## 概要

このアプリは、授業ごとに課題を登録し、登録した授業と課題を一覧で確認できるコンソールアプリである。
Pythonを用いて作成しており、メニューから授業登録、課題登録、一覧表示、終了を選択して操作する。

## 目的

授業ごとの課題を管理し、どの授業にどの課題があるかを簡単に確認できるようにすることを目的とする。

## 使用言語

* Python

## 主な機能

### 1. 授業登録機能

授業名を入力し、授業一覧に登録する。
授業名が未入力の場合は、エラーメッセージを表示して登録を行わない。

### 2. 課題登録機能

登録済みの授業を一覧表示し、課題を登録する授業を番号で選択する。
選択した授業に課題名を追加する。

以下の場合はエラーメッセージを表示する。

* 授業が1件も登録されていない場合
* 授業番号が未入力の場合
* 授業番号に数字以外が入力された場合
* 存在しない授業番号が入力された場合
* 課題名が未入力の場合

### 3. 一覧表示機能

登録されている授業と、その授業に登録されている課題を一覧表示する。
課題が登録されていない授業については「課題なし」と表示する。

### 4. 終了機能

メニューで「4」を選択すると、プログラムを終了する。

## データ構造

授業情報はリスト `subjects` で管理する。
各授業は辞書型で管理し、授業名と課題一覧を持つ。

```python
subjects = [
    {
        "name": "情報数学",
        "assignments": ["レポート1", "レポート2"]
    }
]
```

各項目の意味は以下の通りである。

| 項目          | 内容             |
| ----------- | -------------- |
| name        | 授業名            |
| assignments | その授業に登録された課題一覧 |

## 操作方法

プログラムを実行すると、以下のメニューが表示される。

```text
==== メニュー ====
1. 授業登録
2. 課題登録
3. 一覧表示
4. 終了
選択してください:
```

番号を入力することで、各機能を実行できる。

## エラー処理

このアプリでは、入力内容が正しくない場合にエラーメッセージを表示する。

| 状況                 | 表示メッセージ                         |
| ------------------ | ------------------------------- |
| 授業名が未入力            | エラー：授業名を入力してください。               |
| 授業が未登録の状態で課題登録を行った | エラー：授業が登録されていません。先に授業を登録してください。 |
| 授業番号が未入力           | エラー：授業番号を入力してください。              |
| 授業番号に数字以外を入力       | エラー：数字を入力してください。                |
| 存在しない授業番号を入力       | エラー：存在しない授業番号です。                |
| 課題名が未入力            | エラー：課題名を入力してください。               |
| メニューで1～4以外を入力      | エラー：1～4を入力してください。               |

## 関数一覧

| 関数名                   | 内容                |
| --------------------- | ----------------- |
| register_subject()    | 授業を登録する           |
| register_assignment() | 登録済みの授業に課題を登録する   |
| show_list()           | 授業ごとの課題一覧を表示する    |
| main()                | メニューを表示し、各機能を呼び出す |
