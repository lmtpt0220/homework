# 授業課題管理アプリ

## 概要

このアプリは、授業と課題を管理するためのWebアプリです。  
月曜〜土曜、1限〜6限の時間割表に授業を登録し、各授業に課題を追加できます。

登録した内容はブラウザ内に保存されるため、ページを更新しても同じブラウザであればデータが残ります。

## 公開方法

このアプリは、GitHub Pagesで公開することを想定しています。  
HTML、CSS、JavaScriptのみで作成しているため、サーバーやPython環境は不要です。

## 使用技術

- HTML
- CSS
- JavaScript
- localStorage

## 主な機能

### 1. 授業登録機能

曜日、時限、授業名を入力して授業を登録できます。

登録できる曜日は以下の通りです。

- 月曜
- 火曜
- 水曜
- 木曜
- 金曜
- 土曜

登録できる時限は以下の通りです。

- 1限
- 2限
- 3限
- 4限
- 5限
- 6限

授業名が空欄の場合は、エラーメッセージを表示して登録しません。  
また、同じ曜日・同じ時限にすでに授業が登録されている場合も、重複登録を防ぐためエラーを表示します。

### 2. 課題登録機能

登録済みの授業を選択し、その授業に課題を追加できます。

授業が1件も登録されていない場合は、課題を登録できません。  
また、課題名が空欄の場合は、エラーメッセージを表示して登録しません。

### 3. 授業削除機能

登録済みの授業を選択して削除できます。

授業を削除すると、その授業に登録されている課題も同時に削除されます。  
削除後は、時間割表、課題登録用の授業選択欄、授業・課題一覧が自動で更新されます。

### 4. 時間割表表示機能

月曜〜土曜、1限〜6限の時間割表を表示します。

登録した授業は、対応する曜日・時限のマスに表示されます。  
また、各授業に登録されている課題数も表示されます。

表示例：

```text
情報数学
課題: 2件
```

### 5. 授業・課題一覧表示機能

登録されている授業と課題を一覧で確認できます。

各授業について、以下の情報を表示します。

- 曜日
- 時限
- 授業名
- 登録されている課題一覧

課題が登録されていない授業については、「課題なし」と表示します。

### 6. データ保存機能

ブラウザの `localStorage` を使用して、授業と課題のデータを保存します。

そのため、ページを更新しても同じブラウザであれば登録内容が残ります。  
ただし、別の端末や別のブラウザとはデータを共有できません。

## データ構造

授業データは、以下のような配列で管理しています。

```javascript
[
  {
    "day": "月曜",
    "period": "1限",
    "name": "情報数学",
    "assignments": ["レポート1", "レポート2"]
  }
]
```

各項目の意味は以下の通りです。

| 項目 | 内容 |
|---|---|
| day | 授業の曜日 |
| period | 授業の時限 |
| name | 授業名 |
| assignments | その授業に登録された課題一覧 |

## 画面構成

アプリの画面は、以下のエリアで構成されています。

### 授業登録エリア

授業を登録するためのエリアです。

入力項目：

- 曜日
- 時限
- 授業名

### 課題登録エリア

登録済みの授業に課題を追加するためのエリアです。

入力項目：

- 課題を登録する授業
- 課題名

### 授業削除エリア

登録済みの授業を削除するためのエリアです。

入力項目：

- 削除する授業

### 時間割表エリア

登録済みの授業を時間割形式で表示します。

### 授業・課題一覧エリア

授業ごとの課題を一覧で表示します。

## エラー処理

このアプリでは、以下の場合にエラーメッセージを表示します。

| 状況 | 表示メッセージ |
|---|---|
| 授業名が空欄 | エラー：授業名を入力してください。 |
| 同じ曜日・時限に授業が登録済み | エラー：この曜日・時限にはすでに授業が登録されています。 |
| 授業未登録の状態で課題登録 | エラー：授業が登録されていません。先に授業を登録してください。 |
| 課題名が空欄 | エラー：課題名を入力してください。 |
| 授業未登録の状態で削除 | エラー：削除できる授業がありません。 |

## 使い方

### 1. 授業を登録する

1. 曜日を選択する
2. 時限を選択する
3. 授業名を入力する
4. 「授業を登録する」ボタンを押す

### 2. 課題を登録する

1. 課題を登録したい授業を選択する
2. 課題名を入力する
3. 「課題を登録する」ボタンを押す

### 3. 授業を削除する

1. 削除したい授業を選択する
2. 「授業を削除する」ボタンを押す

### 4. 時間割を確認する

登録した授業は、時間割表に自動で表示されます。  
授業ごとの課題数も確認できます。

### 5. 授業・課題一覧を確認する

登録されている授業と、その授業に登録されている課題を一覧で確認できます。

## 特徴

- GitHub Pagesで公開できる
- HTML、CSS、JavaScriptだけで動作する
- サーバーやデータベースが不要
- 時間割形式で授業を確認できる
- 授業ごとに課題を管理できる
- 授業を削除すると、関連する課題も削除される
- localStorageにより、ブラウザ内にデータを保存できる

## 注意点

このアプリは、ブラウザの `localStorage` にデータを保存しています。  
そのため、以下の点に注意が必要です。

- 別のPCやスマートフォンとはデータを共有できません
- 別のブラウザでは同じデータを確認できません
- ブラウザのデータを削除すると、登録した授業や課題も消えます
- ログイン機能やクラウド保存機能はありません
